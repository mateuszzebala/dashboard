# Dashboard for server in django and react

## How to run django app?

`cd django`

`pip install -r requirements.txt`

`python manage.py runserver 0.0.0.0:8000`

## How to run dev react app?

`cd dashboard/app`

`yarn install`

`yarn start`

## How to build react app?

`cd dashboard/app`

`yarn install`

`yarn run build`
